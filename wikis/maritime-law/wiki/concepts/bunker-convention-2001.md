# Bunker Convention 2001 — Bunker Fuel Spill Liability

## Summary

| Property | Value |
|----------|-------|
| Scope | Bunker fuel spills from non-tankers |
| Key Feature | Strict liability registered owner, compulsory insurance |
| Reference | IMO LEG/CONF.12/19 |

## Details

Strict liability registered owner, compulsory insurance.
