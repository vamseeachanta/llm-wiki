# API RP 11V6 — Gas Lift Design and Technology

## Scope

API RP 11V6 is the canonical gas-lift design reference. Covers the engineering workflow for designing a continuous or intermittent gas-lift installation — valve selection, valve-spacing depth calculations, surface-injection-pressure budget, kickoff procedure, and the design-vs-operating-conditions diagnostic envelope.

## Where it applies

Operator production engineers designing new gas-lift completions; service-company applications engineers validating designs and proposing tuning changes for declining wells.
