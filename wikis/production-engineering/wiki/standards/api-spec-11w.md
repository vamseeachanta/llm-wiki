# API Spec 11W — Progressing-Cavity Pumping Systems

## Scope

API Spec 11W specifies design and testing requirements for progressing-cavity pump (PCP) systems — both surface-driven (rod-string-rotated, like a sucker-rod-pumping system but with rotating rod string) and electric-submersible PCP (ESPCP, where a downhole motor turns the rotor directly). Covers rotor/stator geometry, elastomer-stator chemistry families, performance-testing protocols.

## Where it applies

PCP manufacturers (NOV, Weatherford, Halliburton ALS, Robbins & Myers / NOV historical) seeking API monogram licensing. Operator critical-service procurement for heavy-oil and sand-bearing applications.
