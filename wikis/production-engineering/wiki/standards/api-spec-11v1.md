# API Spec 11V1 — Gas Lift Valve Specification

## Scope

API Spec 11V1 specifies manufacturing requirements for gas-lift valves — bellows-charged and spring-loaded variants, dummy valves, latching mechanisms, and the side-pocket / tubing-retrievable mandrel interface. Covers materials, dimensions, pressure ratings, and corrosion-service variants for sour environments.

## Valve families covered

- **Injection-Pressure-Operated (IPO)** — opens on rising casing-side pressure; bellows charged with nitrogen
- **Production-Pressure-Operated (PPO)** — opens on rising tubing-side pressure; less common but useful for unloading
- **Spring-loaded** — alternative to bellows-charged for higher-temperature service
- **Dummy valves** — blanks installed in mandrels where active valves aren't yet needed
- **Continuous-flow vs intermittent** — orifice sizing differs significantly

## Where it applies

Gas-lift valve manufacturers seeking API monogram; operator critical-service procurement specifications.
