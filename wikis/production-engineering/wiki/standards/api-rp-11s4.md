# API RP 11S4 — ESP Sizing and Selection

## Scope

API RP 11S4 is the canonical reference for sizing an ESP system to a well. Walks the design engineer through the iterative sizing workflow: inflow performance relationship (IPR) → required total dynamic head (TDH) → pump-stage count → motor sizing → cable sizing → VFD selection → seal-section selection. Conservatively, an over-sized ESP cycles inefficiently and under-sized ESP under-produces; RP 11S4 codifies the engineering tradeoff.

## Sizing-workflow inputs

- Reservoir IPR (productivity index, bubble-point, drawdown limit)
- Required production rate
- Wellbore deviation profile (affects intake placement and gas-handling)
- Fluid properties (oil API, water cut, GOR, free-gas-at-intake fraction)
- Casing / tubing size (sets max ESP OD)
- Power supply (surface voltage, available phase, frequency)

## Where it applies

Operator design engineers sizing new ESP completions; ESP-service-companies validating operator-supplied sizing for warranty / performance commitments.
