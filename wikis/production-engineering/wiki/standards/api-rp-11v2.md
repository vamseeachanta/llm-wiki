# API RP 11V2 — Gas Lift Valve Performance Testing

## Scope

API RP 11V2 specifies the laboratory test methodology for gas-lift valves — opening / closing pressure response, flow-coefficient characterization, response under upstream pressure variations, and temperature-effect characterization. Output is the valve performance data that designers consume via [API RP 11V6](api-rp-11v6.md).

## Key test outputs

- **Opening pressure** at design temperature (the surface-injection pressure threshold that lifts the bellows or moves the spring stem)
- **Closing pressure** (typically lower than opening due to flow-induced forces)
- **Flow coefficient** (Cv) over the operating range
- **Temperature correction factor** — bellows-charge gas pressure varies with temperature, shifting opening pressure
- **Stem-travel characteristic** — how the orifice opens vs valve travel

## Where it applies

Gas-lift valve manufacturers qualifying valves for API monogram licensing; operator critical-service acceptance testing.
