# API RP 11S — Operation, Maintenance, and Troubleshooting of ESP Systems

## Scope

API RP 11S is the practitioner-canonical reference for ESP-system field operation, maintenance intervals, and field-failure diagnostics. Covers start-up procedure, normal operating envelope (current vs voltage curves), pull-down procedures, common failure-mode signatures (electrical-fault, mechanical-fault, fluid-related), and troubleshooting decision-trees.

## Where it applies

Field operations crews running ESP wells; ESP-service-company engineers troubleshooting downhole problems via surface VFD data; operator dashboard-monitoring of ESP fleets.
