# API RP 11S2 — Electrical Submersible Pump Testing

## Scope

API RP 11S2 specifies the laboratory and shop-floor performance-testing methodology for ESP pump-stage assemblies and complete pump-stage strings. Output is a head-vs-flow performance curve (similar to surface centrifugal-pump testing) corrected to standard reference conditions, used by operators for sizing-verification.

## Test outputs

- Head-vs-rate curve per pump model at design RPM (typically 3,500 RPM)
- Efficiency curve over the operating range
- Brake horsepower curve
- NPSH-required curve (intake suction-pressure requirement)
- Best-efficiency point (BEP) and operating-window limits

## Where it applies

ESP manufacturers qualifying pump-stage models for API monogram licensing; operator-conducted acceptance testing for critical-service deployments.
