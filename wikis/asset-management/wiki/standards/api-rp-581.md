# API RP 581 — Risk-Based Inspection Methodology (Quantitative)

## Scope

API RP 581 is the quantitative implementation of API RP 580. It supplies damage-mechanism factors, generic failure frequencies, probability-of-failure (PoF) computations, consequence-of-failure (CoF) computations, and inspection-effectiveness tables used to derive risk numerically.

## Where it applies

Operators who require a numerical risk basis for inspection-interval setting, life-cycle cost optimization, or insurance/audit-defensible documentation. Typically deployed via specialist software (RBMI, Synergi Plant, etc.).
