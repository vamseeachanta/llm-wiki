# API 579-1 / ASME FFS-1 — Fitness-for-Service

## Scope

API 579-1 / ASME FFS-1 is the joint API/ASME engineering standard for evaluating the structural integrity of in-service pressure equipment containing flaws, damage, or operating outside the original design envelope. Levels 1–3 provide tiered analysis approaches from screening through full numerical assessment.

## Where it applies

Whenever an inspection campaign (driven by RBI or anomaly management) finds a defect that exceeds construction-code acceptance limits but the operator wants to defensibly continue service, perform a run/repair/replace decision, or extend life.
