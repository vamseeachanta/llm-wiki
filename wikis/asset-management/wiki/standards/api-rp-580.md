# API RP 580 — Risk-Based Inspection

## Scope

API RP 580 documents the methodology for applying risk-based inspection (RBI) to fixed equipment in process plants — pressure vessels, piping, storage tanks, and pressure-relief devices. It is the qualitative / semi-quantitative framework; API RP 581 supplies the quantitative companion.

## Where it applies

Process facilities (refining, petrochemical, upstream oil & gas), where inspection planning must be defensibly risk-prioritized. Frequently combined with DNV-RP-G101 for offshore-topsides scope.
