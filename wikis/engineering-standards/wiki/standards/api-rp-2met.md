# API RP 2MET — Metocean design and operating conditions

## Scope

API recommended practice for the derivation and use of metocean design and
operating conditions applicable to offshore structures (winds, waves,
currents, water levels, return-period methodology). First edition (November
2014). Bounded metadata-only page; no tabulated values or hindcast outputs
reproduced.
