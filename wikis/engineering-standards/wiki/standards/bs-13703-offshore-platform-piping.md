# BS EN ISO 13703 — Design and installation of piping systems on offshore production platforms

## Scope
Bounded resolver target for the BSI-published 2001 adoption of ISO 13703 covering design and installation of piping systems on offshore production platforms. Addresses material selection, design analysis, fabrication, examination, testing, and installation requirements for offshore process piping. The 2001 BS edition mirrors the original ISO 13703 monolithic structure; the ISO standard has since been re-issued as a multi-part series (ISO 13703-1/-2/-3, 2023 cycle).

## Why this page exists
This page is a citation resolver target for downstream calc modules per `.claude/rules/calc-citation-contract.md`. The 2001 BS form is the BSI-published adoption of the original ISO 13703; cite the current ISO multi-part `code_id` (`iso-13703-3` for offshore process piping) for current technical equivalence and this BS page for the historic 2001 edition's UK jurisdictional traceability. Calc-callers using the 2001 BS form should be aware they are citing a historic edition relative to ISO's current multi-part structure. Page contains no clause text or table reproductions.
