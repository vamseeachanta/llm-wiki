## Scope

ISO 9223 is the **international classification framework for atmospheric corrosivity**. It defines six corrosivity categories — **C1 (very low), C2 (low), C3 (medium), C4 (high), C5 (very high), and CX (extreme)** — that characterise the aggressiveness of an outdoor or sheltered atmosphere toward unprotected carbon steel, zinc, copper, and aluminium. The standard provides two complementary derivation pathways: (1) **measurement-based** classification using one-year mass-loss exposure of standard reference specimens (per the companion ISO 9226 method), and (2) **classification-from-environment** using ranges of the controlling environmental parameters — Time of Wetness (TOW), chloride deposition rate, and sulfur dioxide deposition / concentration — when reference-specimen exposure is impractical.

ISO 9223 is the load-bearing input to international coating-system selection, offshore-zone designation, and atmospheric maintenance-interval planning across the steel-structures, oil-and-gas, marine, infrastructure, and renewable-energy industries.

## Revision history

- **ISO 9223:1992** — original publication; defined the C1–C5 framework using qualitative environmental descriptors and reference-specimen exposure.
- **ISO 9223:2012** — comprehensive revision; added the **CX (extreme)** category for tropical-marine, industrial-marine, and similar aggressive environments; tightened deposition-class boundaries (τ / S / P classes); refined the dose-response equations relating environment to first-year corrosion rate; harmonised with ISO 9224 / 9225 / 9226 as a coherent four-part family.
- Subsequent revisions — consult the ISO catalog for the current edition before normative citation.

## Key sections

- **Categorisation parameters** — Time of Wetness (τ classes τ1–τ5 in hours-per-year above ~80 % RH at the metal surface); chloride deposition rate (S classes S0–S3 in mg Cl⁻ /m²/day per ISO 9225); sulfur dioxide deposition / concentration (P classes P0–P3 per ISO 9225); supplementary climate descriptors (mean annual temperature, RH).
- **Classification methodology** — measurement-based pathway using one-year reference-specimen mass-loss per ISO 9226 (carbon steel + zinc + copper + aluminium); environment-based pathway using TOW × deposition class lookup tables; reconciliation guidance when the two pathways disagree.
- **Annex B (environmental-parameter classification)** — the lookup table from environmental-parameter classes to corrosivity category used when reference-specimen exposure is impractical.
- **Annex C (first-year corrosion rates per category)** — the numerical first-year rate ranges that define category boundaries for carbon steel, zinc, copper, and aluminium.
- **Companion standards** — ISO 9224 supplies long-term corrosion-rate guiding values per category; ISO 9225 supplies the deposition-rate measurement methods; ISO 9226 supplies the reference-specimen exposure procedure.

## Practitioner application

- **Coating selection** — ISO 12944 paint-system specification matrices key directly to the ISO 9223 corrosivity category to choose primer / mid-coat / top-coat ladder and DFT (dry-film thickness) tier; a C4 environment requires a substantially heavier paint system than a C2 environment.
- **Offshore / onshore zone designation** — splash-zone, atmospheric-zone, and immersed-zone boundaries on offshore platforms, FPSOs, and coastal terminals are characterised against ISO 9223 categories (typically C5–CX in the splash zone, C3–C4 inland).
- **Maintenance-interval planning** — re-coat intervals for structural steel, transmission towers, bridges, and offshore topsides are scaled against the assigned ISO 9223 category, with shorter intervals at C5/CX and longer intervals at C2/C3.
- **First-year corrosion-rate sanity-checking** — the Annex C rate ranges are used as a band to validate measured wall-loss rates from CMLs (condition-monitoring locations) and atmospheric corrosion coupons.

## Industry adoption

ISO 9223 is the global default for atmospheric corrosivity classification. Key downstream and parallel standards:

- **ISO 12944** (paint-system specification for protection of steel structures) keys its system-selection matrix directly to ISO 9223 corrosivity categories.
- **ISO 9224 / 9225 / 9226** are the companion standards in the four-part atmospheric-corrosion family (long-term rates / deposition measurement / reference-specimen exposure).
- **AMPP / SSPC / NACE** US corrosion-engineering standards reference ISO 9223 categories for atmospheric-environment characterisation; AMPP TM-0204 atmospheric-coupon testing and SSPC paint-system specifications operate in the same parameter space.
- **ASTM G50** is the US-side outdoor-exposure practice that, together with ASTM G84 (TOW) and ASTM D610 (rust-rating), forms the parallel US methodology for populating the same atmospheric-corrosion parameter space.
