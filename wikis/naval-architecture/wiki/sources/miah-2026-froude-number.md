# Miah (2026) — Froude Number Overview

Raw URL: https://www.linkedin.com/posts/hasan-miah-2a9301100_marineengineering-navalarchitecture-froudenumber-share-7459941094001135616-Lesu

Author: Hasan Miah. Format: LinkedIn post (short practitioner explainer). Date observed: 2026-05-12.

## Relevance

A compact restatement of the Froude number definition and its operational applications for naval architects and ship hydrodynamicists. Useful as a popular-explainer trigger source for the [Froude Number](../concepts/froude-number.md) concept page; the concept-page treatment substitutes the post's single-regime framing with the displacement / semi-displacement / planing regime spectrum grounded in textbook references.

## Key teachings (as presented by the author)

- Definition: Fn = V / sqrt(g·L), with V = ship speed, g = gravitational acceleration, L = characteristic length (typically waterline length).
- Application 1 — wave resistance prediction.
- Application 2 — hull design and model testing.
- Application 3 — performance comparison across vessel sizes (similitude).
- Application 4 — towing-tank experiments and CFD analysis.
- Application 5 — fuel efficiency and operational performance.

## Limitations of this source

The post does not distinguish operating regimes (displacement / semi-displacement / planing), nor variants (volumetric Froude number Fn∇, depth Froude number Fnh), nor the practical limit that Froude and Reynolds similitude cannot be matched simultaneously at model scale. A commenter (Abrar Jahin) raised the regime-distinction omission. The concept page covers these explicitly with public references.
