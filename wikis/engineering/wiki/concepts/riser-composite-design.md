# Composite Riser — Design

## Scope

This page covers fibre-reinforced-polymer (FRP) composite riser design: layered material architecture, hybrid metallic-composite construction, weight savings, and qualification. It is NOT an unbonded-flexible page — see [[riser-flexible-design]] for layer architecture of conventional flexibles. It is NOT a metallic riser page — see [[riser-steel-catenary-design]] and [[riser-top-tensioned-design]].

## Key Concepts

- **Composite riser rationale** — high strength-to-weight ratio reduces top tension and enables ultra-deepwater applications where steel weight becomes prohibitive.
- **Fibre architecture** — typically carbon-fibre / epoxy laminate with hoop, axial, and helical fibre orientations to react internal pressure, axial tension, and torsion separately.
- **Liner** — internal metallic or thermoplastic liner forming the fluid barrier; the composite shell carries structural load.
- **Metal-composite end fittings** — composite tube terminated by titanium or steel end fittings; load-transfer joint design is qualification-critical.
- **Damage tolerance** — impact damage and barely-visible-impact-damage (BVID) detection drives in-service inspection requirements.
- **Long-term degradation** — moisture ingress, fibre-matrix interface ageing, and creep set service-life limits.
- **Qualification** — extensive prototype testing per recommended practice; first-of-a-kind qualification programmes are multi-year.
- **Hybrid configurations** — composite mid-section with steel hang-off and TDP joints to localise composite use to depth-driven weight regions.
