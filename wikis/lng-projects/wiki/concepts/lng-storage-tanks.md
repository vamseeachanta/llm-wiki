# LNG Storage Tanks

## Scope

This page summarizes the canonical taxonomy of onshore and floating LNG storage containment systems. It does **not** enumerate clause-level design rules, allowable stress tables, foundation heave limits, or specific safety-distance formulas; those are reserved for future standards-page authoring once routing for this wiki is sanctioned.

## Key Concepts

- **Single-containment** — a single primary container; the outer wall is not designed for cryogenic or vapor containment in a leak scenario; rarely used for new construction
- **Double-containment** — primary inner tank plus a structural outer wall capable of holding liquid in a leak; outer wall not vapor-tight
- **Full-containment** — primary inner tank plus a self-supporting concrete outer tank designed to hold both liquid and vapor on a primary failure; the prevailing onshore choice
- **Membrane tanks** — thin metallic membrane backed by load-bearing insulation; common in floating and some onshore service
- **In-ground tanks** — concrete tank set below grade; favored in seismically active urban-adjacent sites
- **Self-supporting prismatic / Moss-type spherical** — common floating storage containment families used on LNG carriers and on some FLNG units
- **Boil-off rate (BOR)** — typical design BOR is a fraction of inventory per day; managed by reliquefaction or fuel-gas use

## Standards / References

- [NFPA 59A](../standards/nfpa-59a.md) — US-jurisdiction LNG production/storage/handling design code governing onshore tank containment categories
- [IGC Code](../standards/igc-code.md) — IMO international gas-carrier code governing ship-side tank types (membrane, Moss-spherical, prismatic) on LNG carriers and FLNG hulls
- [API Std 625](../standards/api-std-625.md) — Tank Systems for Refrigerated Liquefied Gas Storage; the umbrella tank-system standard whose containment-category taxonomy (single / double / full / membrane) this concept page mirrors
- [CSA Z276](../standards/csa-z276.md) — Canadian LNG production / storage / handling standard whose Clause 5 governs onshore tank containment design in Canadian jurisdictions
- [EN 1473](../standards/en-1473.md) — European harmonized standard for onshore LNG installations; Clause 7 delegates cryogenic tank design to EN 14620 (the EU parallel to API 625)
- EEMUA Publication 147 / 159 (LNG storage tank in-service inspection): <https://www.eemua.org/>
