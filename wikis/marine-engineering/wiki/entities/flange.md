# Flange (Piping)

A mechanical connector used to join pipe sections, valves, pumps, and other equipment in process
piping systems. Allows for assembly, disassembly, and maintenance access.

## Common Types

| Type | Application | Weld Required |
|------|-------------|---------------|
| Welding Neck (WN) | High-pressure, critical service | Butt weld |
| Slip-On (SO) | Low to moderate pressure | Fillet weld (inside + outside) |
| Blind | End closure, blanking | None (bolted) |
| Socket Weld (SW) | Small-bore piping (< 2") | Fillet weld |
| Threaded (THD) | No welding, special services | None |

## Pressure-temperature Ratings

ASME B16.5 standard covers flanges NPS 1/2 through 24:
- Class 150 (lowest) through Class 2500 (highest)
- Temperature derating required above reference temperature
