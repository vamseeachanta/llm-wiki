# Gasket

A sealing element installed between flange faces to prevent fluid leakage from piping joints.

## Common Types

| Type | Material | Application |
|------|----------|-------------|
| Ring Type Joint (RTJ) | Steel (soft iron, SS) | High pressure, RTJ flanges |
| Spiral Wound | Graphite or PTFE filler + metal winding | General process piping |
| Sheet Gasket | Compressed fiber, asbestos-free | Low pressure, non-critical |
| Camprofile | Graphite + serrated metal core | Heat exchangers, critical service |

## Selection Criteria

- **Pressure class**: Higher class → typically RTJ gasket
- **Fluid type**: Hydrocarbon service requires suitable gasket material
- **Temperature**: Gasket material must handle operating temperature
- **Bolt load**: Gasket requires adequate bolt load to achieve seal
