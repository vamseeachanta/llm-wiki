# Classification Societies — Drilling

## Scope

The IACS member classification societies relevant to mobile offshore drilling units (MODUs) — the marine-vessel side of [jackup](../concepts/jackup-rig.md), [semi-submersible](../concepts/semi-submersible-rig.md), and [drillship](../concepts/drillship.md) drilling rigs. Each society publishes MODU-specific rules covering structural integrity, machinery, fire safety, life-saving, and station-keeping that supplement IMO MODU Code requirements.

## IACS member societies active in drilling

| Society | HQ | MODU rules anchor |
|---|---|---|
| **ABS** — American Bureau of Shipping | Houston | ABS Rules for Building and Classing Mobile Offshore Drilling Units |
| **DNV** — Det Norske Veritas | Oslo | DNV Rules for Classification of Offshore Units / DNVGL-OS-C201 (after the 2017-2021 GL merger split) |
| **Lloyd's Register** | London | LR Rules and Regulations for the Classification of Mobile Offshore Units |
| **Bureau Veritas (BV)** | Paris | BV Rules for the Classification of Offshore Units |
| **ClassNK** | Tokyo | NK Rules for Offshore Units (Part U) |
| **KR — Korean Register** | Busan | KR Rules for Mobile Offshore Drilling Units |
| **RINA** | Genoa | RINA Rules for Offshore Units |
| **CCS — China Classification Society** | Beijing | CCS Rules for Offshore Mobile Drilling Units |
| **RS — Russian Maritime Register** | St Petersburg | RS Rules for Offshore Units |
| **IRS — Indian Register of Shipping** | Mumbai | IRS Rules for Mobile Offshore Drilling Units |
| **CRS — Croatian Register of Shipping** | Split | CRS Rules for MODUs |
| **PRS — Polish Register of Shipping** | Gdańsk | PRS Rules for MODUs |

## Cross-domain coverage

Class-society MODU rules cover the marine-vessel side. The drilling-package side is regulated by national petroleum-safety authorities (BSEE in US, HSE in UK, PSA in Norway). The two regimes intersect at the drill floor and BOP-stack handling but are inspected by separate authorities.
