# API Spec 5CT — Specification for Casing and Tubing

## Scope

API Spec 5CT specifies the technical delivery conditions for steel pipes (casing, tubing, and pup joints) used as casing or tubing in oil and natural-gas wells. Covers dimensions, mass, chemistry, mechanical properties, hydrostatic-test requirements, threading inspection, marking, and coating. Identical national adoption of ISO 11960 — cite both `code_id` when applicable to international-jurisdiction work.

## Three Product Specification Levels (PSL)

- **PSL-1** — baseline manufacturing requirements. Default for routine well construction.
- **PSL-2** — adds Charpy V-notch impact testing, hardness limits in HAZ for sour-service grades, stricter NDE.
- **PSL-3** — adds full-body NDE, supplementary chemistry, traceability requirements; specified for critical-service wells (HPHT, deep sour, etc.).

PSL is **operator-specified at purchase**; not all grades are available at all PSL levels.

## Where it applies

All new casing and tubing manufactured to API conformance. Mandatory for API-licensed manufacturers (5CT monogram); operators may specify additional API RP 5C1 (care and use) and API RP 5C5 (connection evaluation) supplements.
