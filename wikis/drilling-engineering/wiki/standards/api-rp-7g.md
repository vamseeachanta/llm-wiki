# API RP 7G — Drill Stem Design and Operating Limits

## Scope

API RP 7G is the canonical drill-stem design reference. Provides minimum-yield, torsional-yield, and collapse-strength values for the API drill-pipe grades; tool-joint makeup-torque tables; pin-and-box tensile capacities; and design-limit envelopes for combined-load operation (tension + torque + bending + internal pressure).

Cited by API Spec 5DP as the performance-properties companion — Spec 5DP fixes how the pipe is made; RP 7G fixes how the pipe is used.

## Design-limit framework

- **Tension** — yield-based for makeup; ultimate-based for overpull
- **Torque** — pipe-body torsional yield; tool-joint makeup torque; tool-joint shoulder yield
- **Collapse** — under fluid-density differential during empty-pipe / partially-evacuated conditions
- **Bending** — dogleg-severity-induced bending stress added to axial tension
- **Combined** — von Mises envelope across all four

## Where it applies

Drill-stem design calculations performed by operator drilling engineers and service-company applications engineers when planning a well or evaluating a directional / extended-reach hole section.
