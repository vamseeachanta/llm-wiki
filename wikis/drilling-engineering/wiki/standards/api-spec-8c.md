# API Spec 8C — Drilling and Production Hoisting Equipment

## Scope

API Spec 8C specifies the technical-delivery requirements for drilling and production hoisting equipment — drawworks, crown blocks, traveling blocks, hooks, hook blocks, deadline anchors, weight indicators, and the supporting Product Specification Family (PSF) classification.

Note: Spec 8C is the successor to the older API Spec 8A (drilling hooks) and API Spec 9A (drilling line / wire rope). Some legacy operator documents still cite the older numbers; the modern reference is 8C.

## PSL framework

Two Product Specification Levels:

- **PSL-1** — baseline
- **PSL-2** — adds full-traceability requirements, supplementary NDE, design-fatigue analysis. Operator-specified for HPHT and critical-well service.

## Where it applies

Hoisting-system manufacturers seeking API monogram licensing; operator specifications when matching rated hoisting capacity to the well's heaviest casing or BHA load.
