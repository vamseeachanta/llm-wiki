# API Spec 7-1 — Specification for Rotary Drill Stem Elements

## Scope

API Spec 7-1 specifies the manufacturing requirements for the **non-pipe** elements of the rotary drill stem — drill collars, heavyweight drill pipe (HWDP), kellys (square and hexagonal), subs (crossover, bit, float), stabilizers, reamers, and drill-bit connection pins. Companion to API Spec 5DP (drill pipe), which covers the pipe-body element only.

## Element coverage

- **Drill collars** — heavy-wall non-magnetic and steel collars used at the bottom of the drill string for weight-on-bit
- **HWDP (heavyweight drill pipe)** — transition element between drill collars and drill pipe; reduces stress concentration at the BHA-to-drill-pipe interface
- **Kelly** — square or hexagonal cross-section pipe driving the drill string from the rotary table (rare on modern top-drive rigs)
- **Subs** — short connection adapters (crossover, bit, float, lift)
- **Stabilizers and reamers** — BHA components that maintain wellbore geometry
- **Drill-bit shank connections** — API-tapered-thread bit-shank designs

## Where it applies

Manufacturers of rotary drill-stem hardware seeking API monogram licensing; operator and drilling-contractor specifications for the BHA components paired with drill pipe under API Spec 5DP.
