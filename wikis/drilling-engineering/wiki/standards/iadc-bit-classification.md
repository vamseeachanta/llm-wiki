# IADC Bit Classification System

## Scope

The industry-canonical drill-bit classification framework comprising:

1. **Roller-cone (tricone) bit classification** — 4-character code describing formation series, formation type within series, bearing / gauge feature, and special features.
2. **Fixed-cutter (PDC / natural-diamond) bit classification** — 4-character code describing body material, cutter density, cutter size, and bit profile.
3. **Dull-grading code (SPE/IADC 23939)** — standardized 8-character format describing used-bit condition for offset-bit analysis and reason-pulled tracking.

## Authority anchors

- **SPE-23937-MS** "The IADC Roller Bit Classification System," SPE/IADC Drilling Conference 1992 — the canonical citation for the roller-cone code.
- **SPE/IADC 23939** "IADC Dull Grading for PDC Drill Bits," 1992 — the dull-grading standard (applies to both PDC and roller-cone with adjusted interpretation).
- **IADC Drilling Manual** chapter on bit classification — practitioner-canonical restatement.

## Why a single standards page rather than separate ones

The roller-cone and PDC classification systems and the dull-grading code are all IADC-canonical structures published across multiple primary documents (SPE-23937, SPE/IADC 23939, IADC Drilling Manual). This page is the standards-anchor; the per-system decoding details and worked examples live on concept pages.
