# API Spec 10A — Well Cements

## Scope

API Spec 10A specifies physical / chemical / performance requirements for cements used in oil and gas wells. Defines the API cement class system (A through H plus special-purpose) that operators specify in their cementing program. Identical adoption of ISO 10426-1.

## Cement classes (high level)

- **Class A** — ordinary Portland-type; shallow / low-temperature applications
- **Class B** — moderate / high-sulfate resistance variant of A
- **Class C** — high-early-strength; rapid-set applications
- **Class D, E, F** — moderate-to-high-pressure / temperature; retarded set
- **Class G** — **most common cement worldwide**; basic Portland chemistry, intended for use with accelerators and retarders to span the full pressure / temperature envelope
- **Class H** — coarser-ground equivalent of Class G; common in US Gulf Coast operations
- **Special-purpose** — pozzolan-blended, fly-ash, silica-flour for HPHT applications

## Where it applies

Well-cement manufacturers seeking API monogram licensing; operator-specified class in the well's cementing program.
