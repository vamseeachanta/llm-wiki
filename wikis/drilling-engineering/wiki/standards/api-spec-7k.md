# API Spec 7K — Drilling and Well Servicing Equipment

## Scope

API Spec 7K specifies the technical-delivery requirements for major drilling-equipment items — mud pumps, swivels, rotary tables, manual tongs, power tongs, slips, elevators, links, and the hydraulic-system pressure-rated components of the circulating, hoisting, and rotating systems.

## Equipment-family coverage

- **Mud pumps** — triplex and quintuplex high-pressure positive-displacement pumps with continuous-duty and intermittent-duty ratings
- **Rotary tables and master bushings** — turntable-driven rotating equipment (largely superseded by top drives but still extant on land rigs)
- **Top drives** — modern-rig rotating-system replacement for rotary table; integral pipe-handling
- **Swivels** — hose-to-kelly / hose-to-top-drive transition; bearing of the hook load while rotating
- **Tongs, elevators, slips, links** — pipe-handling hardware

## Where it applies

Manufacturers of drilling-equipment seeking API monogram licensing; operator and drilling-contractor specifications for major rotating/circulating/hoisting components.
