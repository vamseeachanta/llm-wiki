# API RP 16Q — Marine Drilling Riser System Design + Operation

## Scope

API RP 16Q is the practitioner-canonical reference for designing, selecting, operating, and maintaining marine drilling riser systems. Covers riser stack-up design (joint-by-joint inventory + ordering), tensioner sizing, weak-point analysis (planned-fracture location in emergency-disconnect scenarios), fatigue-life calculations, riser-management software integration, and operational envelope (max vessel offset, wave height, current).

## Key engineering frameworks

- **Tensioning** — sufficient top tension to prevent buckling at the wellhead end while not over-stressing the upper riser joints
- **Weak point design** — the deliberately-weakest joint that fails first if the riser is overloaded, allowing controlled disconnect
- **Fatigue life** — cumulative-damage estimation from vessel motion + wave loading over the riser's service hours
- **Emergency disconnect sequence (EDS)** — the choreographed sequence to release the LMRP and retract the riser if the vessel must abandon location

## Where it applies

Operator drilling-engineering teams designing deepwater riser stack-ups; drilling-contractor pre-mobilization riser-inventory selection; class-society and regulatory review for HPHT deepwater wells.
