# API RP 53 — Blowout Prevention Equipment Systems for Drilling Wells

## Scope

API RP 53 is the practitioner-canonical integrated reference for BOP systems — covers stack arrangement (which preventers in which order), bypass and choke-line geometry, accumulator and control-system integration with the stack hardware, pressure-testing requirements, and recommended maintenance intervals. The integration layer above the component-level specs in [API Spec 16A](api-spec-16a.md) and [API Spec 16D](api-spec-16d.md).

## Stack-arrangement framework

API RP 53 recommends preventer types and order from top to bottom of the stack:

- **Annular preventer** (top) — packs off around any size pipe in the bore; primary well-shut-in element
- **Variable-bore ram (VBR)** — adjustable-bore ram seals around drill pipe of varying sizes
- **Pipe rams** — fixed-bore ram seals around specific OD pipe (drill pipe, drill collar, casing)
- **Shear/blind ram** — shears whatever is in the bore and seals the hole; last-resort well-control
- **Annular preventer** (bottom, on some configurations) — secondary annular

## Testing and maintenance

- **Pre-spud full pressure test** — to RWP
- **Recurring partial-pressure tests** weekly or biweekly per operator procedure
- **Function tests** — every 7 days, exercising every preventer

## Where it applies

Operator drilling-engineering teams specifying BOP-stack requirements; drilling-contractor BOP-maintenance procedures; regulatory inspection (BSEE, HSE, PSA).
