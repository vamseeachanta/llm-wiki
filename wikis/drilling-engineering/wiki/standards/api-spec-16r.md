# API Spec 16R — Marine Drilling Riser Couplings

## Scope

API Spec 16R specifies the design, materials, and testing of marine drilling riser couplings — the load-bearing connections between successive riser joints. Couplings must withstand combined tension + bending + internal pressure + thermal cycling over thousands of make-up / break-out cycles across the riser's service life.

## Coupling families

- **Bolted flange** — older style; multiple bolts around flange perimeter; slow make-up
- **Dog-style / locking-dog** — radial locking dogs engaged by central-cam rotation; fast make-up (~minutes)
- **Breech-block / quick-connect** — automated rotational lock; modern HPHT drillships
- **Hydraulic-actuated** — wedge-locked by hydraulic pressure; specialty deepwater

## Where it applies

Riser-coupling manufacturers seeking API monogram licensing; operator deepwater-program critical-component procurement.
