# API RP 7L — Drilling Equipment Inspection, Maintenance, Repair, and Remanufacture

## Scope

API RP 7L covers the operational-life management of drilling equipment — pre-mobilization inspection, in-service maintenance intervals, repair procedures, and remanufacture (rebuilding) criteria for the equipment families covered by API Spec 7K (drilling and well-servicing equipment) and Spec 7-1 (rotary drill-stem elements). Also covers wireline-equipment and slickline-unit inspection.

## Coverage families

- **Top drive maintenance** — bearing, gear, swivel inspection
- **Mud-pump preventive maintenance** — valves, liners, gears
- **Rotary table and master bushing** — wear inspection
- **Wireline / slickline equipment** — winches, drums, sheaves
- **Subs and crossovers** — thread-wear and connection-fatigue inspection
- **Drill collar and HWDP** — periodic NDE and dimensional check

## Where it applies

Drilling-contractor maintenance programs; third-party equipment-inspection vendors; operator pre-mobilization equipment audits.
