# API RP 5C7 — Coiled Tubing Operations in Oil and Gas Well Services

## Scope

API RP 5C7 covers the operational practice for coiled-tubing (CT) deployment — surface unit setup, injector head force calculations, reeling fatigue management, BOP and stripper integration, well-control during CT operations, and end-of-job inspection requirements. Companion specification for the tube material itself is [API Spec 5ST](api-spec-5st.md).

## Coiled-tubing applications

- **Well intervention** — wellbore cleanout, scale removal, fishing operations
- **Stimulation** — acid pumping, fracturing fluid placement
- **CT drilling** — slim-hole drilling on existing wellbores
- **Logging** — wireline tools deployed via CT in horizontal / high-angle wells
- **Nitrogen displacement** — well unloading, lift-off

## Where it applies

CT service companies (Halliburton, Schlumberger, BJ Services / Baker Hughes, NCS Multistage) seeking conformance; operator audit of CT vendor capability.
