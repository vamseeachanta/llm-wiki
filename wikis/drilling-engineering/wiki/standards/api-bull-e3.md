# API Bulletin E3 — Wellbore Plugging

## Scope

API Bulletin E3 is the US-anchored practitioner guidance for wellbore plugging operations — the procedural reference for setting cement plugs to permanently isolate zones during well abandonment or temporary suspension. Defines plug placement, depth selection, volume calculation, pre-placement testing, and post-set verification.

## Where it applies

US wellbore-plugging operations under BSEE (offshore) and state regulatory frameworks. Internationally referenced where local regulators don't have stricter standards (NORSOK D-010 supersedes for NCS).
