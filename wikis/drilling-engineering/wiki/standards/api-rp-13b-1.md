# API RP 13B-1 — Field Testing of Water-Based Drilling Fluids

## Scope

API RP 13B-1 standardizes the field-test methodology for water-based drilling fluids (WBMs) — measurement procedures for density (mud-balance), viscosity (Marsh funnel and direct-indicating rheometer), gel strengths (10 sec and 10 min), API filtrate / fluid loss (low-pressure / low-temperature API filter press), pH, methylene blue test (MBT) for cation-exchange capacity, sand content, and solids content.

## Test inventory

- **Mud weight** — mud balance, reported in ppg, sg, or psi/ft gradient
- **Marsh funnel viscosity** — outflow time for 1 qt; quick rig-floor check
- **Rheometer** — direct-indicating dial readings at 600/300/200/100/6/3 RPM; computes PV (plastic viscosity), YP (yield point) per Bingham model
- **Gel strengths** — 10-sec and 10-min static gel strength
- **API filter press** — 30-min filtrate volume at 100 psi room temperature
- **HTHP filter press** — high-temperature high-pressure filtrate (300 °F, 500 psi differential)
- **pH** — paper or electrode
- **MBT, sand, solids** — periodic chemical / mechanical content checks

## Where it applies

Rig-floor mud engineers performing routine mud checks (every 4-8 hours typically) and trend monitoring against the well's mud program specification.
