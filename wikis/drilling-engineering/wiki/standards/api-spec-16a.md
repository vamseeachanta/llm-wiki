# API Spec 16A — Drill-Through Equipment

## Scope

API Spec 16A specifies the technical-delivery requirements for drill-through equipment — ram-type blowout preventers (RAM BOPs), annular blowout preventers, hydraulic connectors, drilling spools, and the rated-working-pressure (RWP) classes that anchor BOP-stack design. The companion to [API Spec 16D](api-spec-16d.md) (control systems) and [API RP 53](api-rp-53.md) (integrated BOP system).

## Rated Working Pressure (RWP) classes

- **2,000 psi (2K)** — shallow / low-pressure wells, largely historical
- **3,000 psi (3K)** — moderate-pressure conventional onshore
- **5,000 psi (5K)** — common offshore moderate-pressure
- **10,000 psi (10K)** — workhorse offshore deepwater
- **15,000 psi (15K)** — HPHT applications
- **20,000 psi (20K)** — frontier HPHT (modern subsea, ultra-deep)

## Where it applies

BOP-equipment manufacturers (Cameron, NOV Shaffer, Hydril/GE, Schlumberger Cameron) seeking API monogram licensing; operator and drilling-contractor specifications matching BOP rating to well's expected pressure envelope.
