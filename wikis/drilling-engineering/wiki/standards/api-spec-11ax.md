# API Spec 11AX — Subsurface Sucker Rod Pump Assemblies, Components, and Fittings

## Scope

API Spec 11AX specifies design, materials, and dimensional requirements for subsurface sucker rod pumps and their components — barrels, plungers, valves (traveling and standing), seating assemblies, pull tubes, hold-down assemblies — in commonly-used bore sizes for the sucker-rod-lift artificial-lift method.

## 12-character pump designation

API Spec 11AX establishes a structured 12-character pump-designation code that encodes tubing size, pump bore size, pump type (insert / tubing), barrel type, plunger length, barrel length, and plunger extensions. Decoded in detail on [api-11ax-pump-designation.md](../concepts/api-11ax-pump-designation.md).

## Where it applies

Sucker-rod pump manufacturers seeking API monogram licensing; operator specifications for downhole-pump procurement; well-servicing rig operations during pump replacement / repair.
