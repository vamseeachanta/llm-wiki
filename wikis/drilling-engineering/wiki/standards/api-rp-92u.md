# API RP 92U — Underbalanced Drilling Operations

## Scope

API RP 92U covers underbalanced drilling (UBD) — drilling with bottomhole pressure deliberately below formation pore pressure so that formation fluid flows continuously into the wellbore during drilling. UBD avoids fluid-loss into the formation (critical for protecting depleted reservoirs from skin damage) and enables real-time productivity testing while drilling.

## UBD fluid types

- **Air / nitrogen** — lowest BHP; for hard-rock arid environments
- **Mist** — fine water droplets in air; for moderate hole-cleaning
- **Foam** — surfactant-stabilized air-water mixture; standard UBD fluid
- **Gasified mud** — conventional mud with nitrogen injection
- **Aerated mud** — lower-density continuous-phase with discrete air bubbles

## Where it applies

Operator UBD-program engineers (typically depleted-reservoir + carbonate reservoirs + reservoir-characterization-during-drilling applications); UBD service-company operations; regulatory review for underbalanced-operations safety case.
