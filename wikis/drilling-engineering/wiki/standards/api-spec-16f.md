# API Spec 16F — Marine Drilling Riser Equipment

## Scope

API Spec 16F specifies manufacturing requirements for marine drilling riser equipment — main riser joints (typically 50-90 ft pup-joints + standard joints), tensioner systems, telescopic joints (slip joints), flex joints, ball joints, choke / kill / booster auxiliary lines clamped to the main riser, and the lower marine riser package (LMRP) interface.

## Where it applies

Drilling-riser-equipment manufacturers (Cameron, NOV, Aker Solutions, Drill-Quip) seeking API monogram licensing; operator critical-service specifications for deepwater drilling.
