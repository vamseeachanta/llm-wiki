# API RP 5C5 — Evaluation Procedures for Casing and Tubing Connections

## Scope

API RP 5C5 defines the qualification-test program for tubular connections — both API-monogrammed connections (API Round, API Buttress, Extreme-Line) and proprietary premium connections (VAM, TenarisHydril, Hunting Seal-Lock families). Covers sealability under combined pressure-tension-bending loading, makeup-and-breakout fatigue, thermal cycling, and seal-integrity test acceptance.

## Connection Application Levels (CAL)

The standard ranks tested connections by CAL — increasing severity from CAL I (atmospheric pressure, low tension) through CAL IV (combined HPHT loads with bending). Operators specify a target CAL when selecting connections for a well.

## Where it applies

Connection-qualification programs run by tubular manufacturers prior to commercial release of a new connection design, and operator audits of connection performance for non-API connections used in critical service.
