# API RP 13B-2 — Field Testing of Oil-Based Drilling Fluids

## Scope

API RP 13B-2 specifies field-test methodology for oil-based muds (OBMs) and synthetic-based muds (SBMs). Mirrors [API RP 13B-1](api-rp-13b-1.md) but with OBM-specific additions: electrical stability (ES), HTHP filter press as primary fluid-loss measurement, brine-phase salinity and water content, and oil-water ratio.

## OBM-specific tests

- **Electrical stability (ES)** — voltage breakdown across an electrode pair immersed in mud; correlates with emulsion stability
- **Water content / salinity** — by retort or Karl Fischer titration; OBM water-phase salinity matters for shale-inhibition
- **Oil-water ratio** — measured directly by retort
- **HTHP filtrate** — primary fluid-loss test for OBMs (300-400 °F, 500 psi differential)
- **Rheometer (Fann 35)** — same as WBM but with temperature control needed for valid PV/YP

## Where it applies

OBM / SBM operations (most modern offshore deepwater and HPHT wells use OBM). Routine rig-floor checks plus shore-lab QC for retained samples.
