# API RP 7G-2 — Inspection and Classification of Used Drill Stem Elements

## Scope

API RP 7G-2 defines inspection levels (DS-1 through DS-5 family), procedures, and acceptance criteria for used drill pipe, tool joints, drill collars, kellys, and subs. The output is a **wear-class assignment** (New / Premium / Class 2 / Class 3) that downstream operators consult when deriving derated design-limit values via API RP 7G.

## Inspection coverage

- Visual / dimensional (OD, ID, wall, drift, length)
- Magnetic particle inspection (MPI) on tool-joint sealing surfaces
- Ultrasonic wall-thickness measurement on pipe body
- Hardness testing on suspect tool joints
- Thread-profile gauging on tool-joint connections

## Where it applies

Drill-pipe inspection facilities (the "rig-floor" between drilling jobs) and the third-party inspection vendors that operators contract for pre-mobilization tubular inspection prior to a critical well.
