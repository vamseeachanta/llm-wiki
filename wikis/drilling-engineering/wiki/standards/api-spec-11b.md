# API Spec 11B — Specification for Sucker Rods, Polished Rods, Liners, Couplings, Sinker Bars, Polished-Rod Clamps, and Pumping Tees

## Scope

API Spec 11B specifies dimensions, materials, mechanical properties, and threading requirements for the sucker-rod elements of a sucker-rod-pumping system — sucker rods (typically 25 ft or 30 ft joint lengths), polished rod, pony rods (short adjustment lengths), liners, rod couplings, sinker bars, polished-rod clamps, and pumping tees.

## Rod grades

API Spec 11B grades for sucker rods (paraphrased):

- **Grade C** — AISI 1536 carbon-manganese steel. General service.
- **Grade D** — alloy steel with higher tensile strength. Heavy-duty / deeper wells.
- **Grade K** — nickel-molybdenum alloy steel for corrosive (H2S) service. NACE MR0175 compliant variants.
- **Grade KD** — high-strength K-equivalent for severe corrosive service.

Detailed minimum-yield, tensile, and elongation values per grade in API Spec 11B tables — not transcribed here per the vendor-derivative deny-list.

## Where it applies

Sucker-rod manufacturers seeking API monogram licensing; operator selection of rod grade for new-completion pump installations.
