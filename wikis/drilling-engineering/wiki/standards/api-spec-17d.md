# API Spec 17D — Subsea Wellhead and Christmas Tree Equipment

## Scope

API Spec 17D specifies design, materials, qualification testing, and pressure ratings for subsea wellhead equipment and subsea Christmas trees. Wellhead scope is the drilling-side interface (casing hangers, lockdown rings, tubing hangers); Christmas tree scope crosses into production-engineering. This wiki page anchors the drilling-side perspective; cross-link to production-engineering for tree-side detail when that scope is developed.

## Wellhead components (drilling-side focus)

- **18-3/4" wellhead housing** — the standard subsea wellhead OD; lands and supports successive casing strings
- **Casing hangers** — one per casing string; lock the casing into the wellhead with rated-load capacity
- **Lockdown ring** — secures hangers against unlatching under high differential pressure
- **Tubing hanger** — installed at completion; lands the production tubing string in the wellhead
- **Test plugs** — used during BOP nipple-down + nipple-up operations

## Pressure rating classes

Same RWP class framework as the BOP (per API Spec 16A): 5K / 10K / 15K / 20K psi. Subsea wellheads are specified to match (or exceed) the BOP RWP class for the well.

## Where it applies

Subsea-wellhead manufacturers (Cameron, FMC Technologies, GE Aker Solutions, NOV Drilquip) seeking API monogram licensing; operator deepwater-completion equipment specification.
