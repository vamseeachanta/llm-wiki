# API RP 92M — Managed Pressure Drilling Operations

## Scope

API RP 92M is the canonical recommended practice for managed pressure drilling (MPD) — an adaptive closed-loop drilling technique that uses a rotating control device (RCD), choke manifold, and back-pressure pump to maintain precise bottomhole pressure control during drilling. MPD enables drilling wells with narrow pore-pressure-to-fracture-pressure windows that would be intractable for conventional overbalanced drilling.

## IADC MPD classification (4 categories)

1. **Constant Bottomhole Pressure (CBHP)** — most common; uses surface back-pressure to maintain BHP at setpoint
2. **Pressurized Mud Cap Drilling (PMCD)** — pressurized cap of mud above a sacrificial fluid; for severe-loss-circulation zones
3. **Dual Gradient** — separate fluid gradients above and below seafloor; deepwater narrow-window
4. **Returns Flow Control** — basic closed-loop without full pressure setpoint; intermediate complexity

## Where it applies

Operator drilling-engineering teams designing narrow-window deepwater + HPHT + depleted-reservoir wells; MPD service-company applications engineers; class-society + regulatory review for HPHT MPD operations.
