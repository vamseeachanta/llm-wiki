# API RP 5C1 — Care and Use of Casing and Tubing

## Scope

API RP 5C1 covers field handling, transportation, storage, makeup, and running procedures for casing and tubing supplied under API Spec 5CT. Covers thread-protector practice, drift testing, makeup-torque control, and rig-floor handling to prevent the dent, bend, and connection-damage failure modes that account for a large fraction of in-service tubular leaks.

## Where it applies

Operator and drilling-contractor field crews handling casing and tubing. Specifically the rig-floor running procedure (stabbing, thread cleaning, makeup-torque verification, slip practice) — the steps not covered by the manufacturing spec.
