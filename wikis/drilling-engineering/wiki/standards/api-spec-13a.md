# API Spec 13A — Drilling Fluid Materials

## Scope

API Spec 13A specifies physical and chemical requirements for commercial drilling-fluid additives — barite (weighting agent), bentonite (viscosifier / fluid-loss control), attapulgite (saltwater-resistant viscosifier), hematite (high-density weighting), calcium chloride, lignosulfonate (thinner), and other additives. Sets the manufacturing-quality baseline that mud-engineering specifications build on.

## Where it applies

Drilling-fluid material manufacturers seeking API monogram licensing. Operator-specified PSL where critical-service application requires guaranteed material quality.
