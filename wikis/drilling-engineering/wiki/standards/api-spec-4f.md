# API Spec 4F — Drilling and Well Servicing Structures

## Scope

API Spec 4F specifies requirements and gives recommendations for steel structures used in drilling and well servicing: derricks, masts (including guy-line and service-rig masts), substructures, and crown-block assemblies. Applies to all new structures with manufacturing date after the effective date of the specification.

## Key features of the 5th Edition

- **AISC Allowable-Strength-Design update** incorporated, aligning steel-design methodology with the latest American Institute of Steel Construction code.
- **Structural Safety Level (SSL) framework** — each structure qualified for two SSLs simultaneously: SSL-1 addressing any anticipated load event, SSL-2 addressing any unexpected event. The SSL framework is a risk-management methodology distinguishing routine-operation loads from extreme-event loads.
- **Two Product Specification Levels (PSL)** — operator-specified manufacturing-quality tier.

## Where it applies

Manufacturers of derricks, masts, substructures, and crown-block assemblies seeking API monogram licensing. Operators specifying structural-rating requirements in drilling-tender RFPs.
