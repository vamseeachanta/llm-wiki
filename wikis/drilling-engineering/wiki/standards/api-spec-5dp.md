# API Spec 5DP — Specification for Drill Pipe

## Scope

API Spec 5DP specifies the technical delivery conditions for steel drill pipe — pipe body and integrally upset tool-joint ends. Covers dimensions, mass, chemistry, mechanical properties, hydrostatic-test requirements, NDE, marking, and traceability. The standard is based on the older API Spec 5D and API Spec 7 lineages; performance properties for drill-pipe design reside in the companion [API RP 7G](api-rp-7g.md).

Identical national adoption of ISO 11961 — Petroleum, petrochemical and natural gas industries — Steel drill pipe. Cite both identifiers for international-jurisdiction work.

## Revision velocity

The 2nd Edition shipped May 2020 with subsequent errata in 2020, 2024, and 2025 plus Addendum 1 in 2025 — five revision events in five years. Downstream consumers should verify the specific revision against the manufacturer's certificate.

## Grade families (named, not transcribed)

- **E-75** — 75 ksi minimum yield; routine
- **X-95** — 95 ksi minimum yield
- **G-105** — 105 ksi minimum yield
- **S-135** — 135 ksi minimum yield; high-strength
- **V-150** — 150 ksi minimum yield; ultra-high-strength (deep wells)

Detailed property tables in API Spec 5DP — see textbook restatement (Bourgoyne Ch. 6, Mitchell-Miska) for design-use values.

## Where it applies

All new drill pipe manufactured to API conformance and used in oilfield rotary-drilling operations.
