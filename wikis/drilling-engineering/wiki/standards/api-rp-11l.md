# API RP 11L — Design Calculations for Sucker Rod Pumping Systems

## Scope

API RP 11L provides correlation-based design calculations for conventional-geometry sucker-rod pumping units. The famous **API 11L charts** predict plunger stroke, peak polished-rod load (PPRL), minimum polished-rod load (MPRL), peak gearbox torque, and surface horsepower from nondimensional input groups (Sp/S, Wrf/Skr, Fo/Skr).

## Authoritative-but-aged

The 4th Edition is from June 1988, reaffirmed since. Despite the age:

- **Still authoritative for conventional Class I beam-pump units** with reasonably regular geometry — the empirical correlations remain valid.
- **Superseded for irregular pumping units** (Mark II, RotaFlex, hydraulic long-stroke) by **wave-equation solvers** introduced by Gibbs (1963) and now standard in commercial software (RodStar, QRod, SROD).

## Where it applies

Conventional-unit sucker-rod-pumping design. Field-engineering sizing of polished-rod load and gearbox torque ratings. Educational tool — the 11L correlations are the foundational way to teach sucker-rod-pumping mechanics.
