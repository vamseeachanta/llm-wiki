# API Spec 16D — Control Systems for Drilling Well Control Equipment

## Scope

API Spec 16D specifies the hydraulic control systems that operate the BOP stack. Covers accumulator sizing, hydraulic power-unit (HPU) requirements, control-manifold design, and the response-time performance the system must meet to close ram preventers within specified seconds.

## Control-system architectures

- **Surface BOP** — direct hydraulic control via koomey-style unit on the rig floor; short hose runs from manifold to BOP
- **Subsea BOP — pilot hydraulic** — older subsea control with hydraulic pilot lines down the riser; slow response, large hose bundle
- **Subsea BOP — MUX (multiplexed electro-hydraulic)** — modern subsea control with electric signals down the riser controlling hydraulic valves on the BOP itself; fast response, smaller umbilical

## Accumulator requirements

API Spec 16D mandates accumulator capacity sized to close-and-open the full BOP function set without HPU pumping — protects against power failure. Specific volume requirements depend on stack configuration.

## Where it applies

BOP-control-system manufacturers seeking API monogram licensing; operator specifications for subsea BOP MUX systems on modern semis and drillships.
