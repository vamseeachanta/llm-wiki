# API Spec 5ST — Coiled Tubing

## Scope

API Spec 5ST specifies the technical-delivery requirements for the coiled-tubing material itself — chemistry, mechanical properties, OD / wall-thickness tolerances, heat-treatment, NDE, marking. Coiled tubing is typically delivered in continuous lengths of 5,000-30,000 ft on a single reel.

## CT grades

API Spec 5ST grades (paraphrased):

- **CT-70, CT-80, CT-90** — yield grades in ksi; routine to intermediate-duty
- **CT-100, CT-110, CT-130** — higher-yield grades for deep / HPHT applications

Detailed property tables in API Spec 5ST.

## Where it applies

CT material manufacturers (Tenaris, NOV, Quality Tubing) seeking API monogram licensing.
