# Sucker Rods and Tapered Strings

## Scope

The sucker-rod string — surface to downhole pump — and the tapered-string design rationale. The rod string is the load-bearing structural element of a sucker-rod-pumping system; its design weight, diameter taper, and grade selection set the well's depth and rate capability.

## Rod elements (surface to downhole)

- **Polished rod** — precision-machined ground-and-polished surface rod inside the stuffing box; provides dynamic fluid seal at the wellhead.
- **Pony rods** — 2 ft to 12 ft short adjustment lengths; tune total string length for proper plunger spacing.
- **Sucker rods** — 25 ft or 30 ft joint lengths (most common API standard); the bulk of the string.
- **Couplings** — threaded couplings between sucker rods (API 5/8", 3/4", 7/8", 1", 1⅛", 1¼" diameters).
- **Sinker bars** — heavy weighted bars at the bottom of the string; ensure compression-free downstroke through the fluid column.

## API Spec 11B grades

(Detailed grade properties in API Spec 11B; named without table transcription per the deny-list.)

- **Grade C** — carbon-manganese; general service
- **Grade D** — alloy; heavier-duty
- **Grade K** — Ni-Mo corrosive service; NACE MR0175 variants
- **Grade KD** — high-strength K; severe sour

## Tapered-string design rationale

A uniform-diameter rod string has the maximum stress at the surface (carries entire fluid + rod weight) and lowest stress at the bottom. A tapered string steps down in diameter from surface to bottom:

- Surface section — largest diameter (e.g., 1") to carry peak load
- Middle section — intermediate (e.g., 7/8")
- Bottom section — smallest (e.g., 3/4")

Tapered design reduces total rod-string weight, lowers polished-rod load, reduces gearbox torque, and improves pumping efficiency. The taper-step pattern is computed to keep peak stress roughly uniform along the string.

## Standard taper combinations

Industry shorthand for taper combinations uses the diameter-in-eighths-of-inches separated by hyphens:

- **86-T** = 1" / 7/8" / 3/4" taper
- **76-T** = 7/8" / 3/4" / 5/8" taper
- **65-T** = 3/4" / 5/8" taper (two-section)
