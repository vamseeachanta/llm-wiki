# Wireline Overview

## Scope

Wireline — cable-deployed tool conveyance into the wellbore — is the foundational well-services technology for logging, perforating, plug-setting, fishing, and downhole measurement. Two cable types (slickline mechanical, e-line electric) and two operating environments (open-hole during drilling, cased-hole for production / intervention).

## Cable types

### Slickline

- Single-strand small-OD steel wire (typically 0.108" or 0.125")
- No electrical conductors
- Used for mechanical operations — bottom-hole pressure / temperature sampling with memory tools, plug setting, valve operation
- Lower-cost than e-line; faster trip times

### Electric line (e-line / wireline)

- Multi-conductor armored cable (typically 7/16" OD with 7 conductors)
- Real-time data transmission to surface
- Used for logging, perforating with surface-fire control, formation-pressure testing (FT)

## Cased-hole vs open-hole

### Open-hole logging

During drilling, run between casing-string section ends to log the formation before cementing the next casing. Standard logs: gamma ray, resistivity (induction, laterolog), density, neutron, sonic, photoelectric, caliper.

### Cased-hole logging

After production casing is run and cemented. Standard logs: cement bond log (CBL/VDL), production logging tools (PLT — flow rates by zone), saturation tools (pulsed neutron), corrosion logs.

## Operational anatomy

- **Wireline truck / skid** — cable drum, winch, depth measurement system
- **Pressure-control equipment (PCE)** — wireline BOP, lubricator, stuffing box for live-well operations
- **Logging tools** — strung downhole on the cable end
- **Surface acquisition** — for e-line, computerized data acquisition and real-time display
