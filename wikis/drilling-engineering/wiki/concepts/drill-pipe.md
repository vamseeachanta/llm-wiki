# Drill Pipe

## Scope

Pipe-body characteristics of API drill pipe — nominal sizes, grades, wall thicknesses, length ranges, and the convention by which a "drill pipe spec" identifies a specific item. Tool-joint geometry lives on [tool-joints.md](tool-joints.md); design-limit envelopes live on [drill-stem-design.md](drill-stem-design.md).

## Nominal sizes

Common drill-pipe nominal ODs (paraphrased from API Spec 5DP):

- 2⅜", 2⅞" — small-OD for slim-hole and workover applications
- 3½", 4", 4½" — intermediate-OD common in directional and HPHT work
- 5", 5½" — workhorse sizes for routine drilling
- 5⅞", 6⅝" — large-OD for high-torque / high-flow applications

## Grade families (API Spec 5DP)

| Grade | Minimum yield (ksi) | Typical use |
|---|---|---|
| E-75 | 75 | Routine |
| X-95 | 95 | Higher-load wells |
| G-105 | 105 | Deeper wells, higher tension |
| S-135 | 135 | High-strength routine |
| V-150 | 150 | Ultra-high-strength, deep / ERD wells |

Detailed yield / ultimate / elongation / hardness tables in API Spec 5DP — see [api-spec-5dp.md](../standards/api-spec-5dp.md). Per the vendor-derivative deny-list, exact table values are not transcribed here.

## Length ranges

API standard length ranges:

- **Range 1** — 18 to 22 ft (rarely used on modern rigs)
- **Range 2** — 27 to 30 ft (most common)
- **Range 3** — 38 to 45 ft (long-stand operations; reduces connections per stand)

## Drift diameter

Each drill-pipe size has a specified drift diameter — the maximum-OD ball that will pass through the pipe ID along its full length without binding. Drift requirements derive from running tools, fishing tools, and the smallest-OD BHA component that must pass through.
